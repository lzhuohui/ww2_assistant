# AI产物目录说明

## 目录结构

```
AI产物/
├── 临时文件/          # 临时生成的文件和工具
│   ├── 修复工具/      # 修复BOM等工具
│   └── 测试文件/      # 测试脚本和临时文件
├── 工具/              # 通用工具和脚本
│   ├── 模板/          # 报告、脚本、配置模板
│   └── 脚本/          # 各种辅助脚本
├── 文档/              # 生成的报告和文档
│   └── 报告/          # 各类报告文件
├── README.md         # 本说明文件
└── 索引.json         # 目录索引
```

## 用途

- **临时文件/**: 存放处理过程中产生的临时文件，包括测试脚本、修复工具等
- **工具/**: 存放可重用的工具脚本和模板
- **文档/**: 存放生成的报告和文档

## 清理建议

- 定期清理临时文件目录中的过期文件
- 保留有价值的工具和文档
- 确保索引.json文件及时更新
