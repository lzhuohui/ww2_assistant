# AI产物目录管理方案

## 1. 目录架构设计

### 1.1 标准路径结构

```
AI产物/
├── 文档/                    # 文档类文件
│   ├── 报告/               # 各类报告文档
│   ├── 方案/               # 设计方案文档
│   ├── 分析/               # 分析文档
│   └── 临时/               # 临时文档
├── 工具/                   # 工具脚本
│   ├── 脚本/               # Python/Bash脚本
│   ├── 配置/               # 配置文件模板
│   └── 模板/               # 代码模板
├── 数据/                   # 数据文件
│   ├── 原始/               # 原始数据
│   ├── 处理/               # 处理后数据
│   └── 备份/               # 数据备份
├── 日志/                   # 日志文件
│   ├── 操作/               # 操作日志
│   ├── 错误/               # 错误日志
│   └── 审计/               # 审计日志
├── 临时/                   # 临时文件
│   ├── 缓存/               # 缓存文件
│   └── 草稿/               # 草稿文件
└── 归档/                   # 归档文件
    ├── 历史/               # 历史版本
    └── 废弃/               # 废弃文件
```

### 1.2 命名规范

#### 文件命名规则
```
[类型]_[日期]_[描述]_[版本].[扩展名]

示例：
- 报告_20260308_环境配置_v1.0.md
- 脚本_20260308_字符统计_v1.0.py
- 数据_20260308_测试结果_v1.0.json
```

#### 目录命名规则
- 使用中文名称，简洁明了
- 避免使用特殊字符和空格
- 统一使用驼峰命名或下划线分隔

## 3. 版本控制规则

### 3.1 版本号规范

```
主版本号.次版本号.修订号

示例：v1.2.3
- 主版本号：重大变更
- 次版本号：功能新增
- 修订号：bug修复
```

### 3.2 版本管理流程

1. **初始版本**：v1.0.0
2. **小修改**：修订号+1（v1.0.1）
3. **功能新增**：次版本号+1（v1.1.0）
4. **重大变更**：主版本号+1（v2.0.0）

### 3.3 版本历史记录

每个文件应包含版本历史：

```markdown
## 版本历史

| 版本 | 日期 | 修改内容 | 作者 |
|------|------|---------|------|
| v1.0.0 | 2026-03-08 | 初始版本 | AI |
| v1.0.1 | 2026-03-09 | 修正错误 | AI |
```

## 5. AI自动识别机制

### 5.1 目录识别规则

AI在生成文件前，自动识别：

```python
def 识别目标目录(文件类型, 文件内容):
    """根据文件类型和内容识别目标目录"""
    
    类型映射 = {
        ".md": "文档/",
        ".py": "工具/脚本/",
        ".json": "数据/",
        ".log": "日志/",
        ".tmp": "临时/",
    }
    
    # 根据内容进一步细分
    if "报告" in 文件内容:
        return "文档/报告/"
    elif "脚本" in 文件内容:
        return "工具/脚本/"
    
    return 类型映射.get(文件扩展名, "临时/")
```

### 5.2 自动路径生成

```python
def 生成文件路径(文件类型, 文件描述, 扩展名):
    """生成标准文件路径"""
    
    当前日期 = datetime.now().strftime("%Y%m%d")
    版本号 = "v1.0.0"
    
    文件名 = f"{文件类型}_{当前日期}_{文件描述}_{版本号}.{扩展名}"
    目标目录 = 识别目标目录(文件类型, 文件描述)
    
    return f"AI产物/{目标目录}{文件名}"
```

## 6. 项目根目录保护

### 6.1 禁止行为

AI不得在以下位置创建文件：
- 项目根目录（除特定配置文件外）
- 源代码目录
- 测试目录
- 文档根目录

### 6.2 自动检查机制

每次文件操作前，AI自动检查：

```python
def 检查路径合法性(目标路径):
    """检查目标路径是否在允许范围内"""
    
    允许前缀 = ["AI产物/", "前端/", "后端/", "测试/"]
    禁止目录 = ["./", "src/", "docs/", ".git/"]
    
    # 检查是否在允许范围内
    if not any(目标路径.startswith(前缀) for 前缀 in 允许前缀):
        return False, "路径不在允许范围内"
    
    # 检查是否在禁止目录
    if any(禁止 in 目标路径 for 禁止 in 禁止目录):
        return False, "路径在禁止目录中"
    
    return True, "路径合法"
```
