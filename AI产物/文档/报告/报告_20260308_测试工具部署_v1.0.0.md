## Trae IDE联动配置

### 1. VS Code配置 (.vscode/)

#### launch.json - 调试配置

提供7种调试配置：
1. **Python: 当前文件** - 调试当前Python文件
2. **Python: 运行模块** - 运行前端入口模块
3. **pytest: 当前文件** - 运行当前文件测试
4. **pytest: 当前目录** - 运行当前目录测试
5. **pytest: 全部测试** - 运行全部测试（带覆盖率）
6. **pytest: 单元测试** - 仅运行单元测试
7. **pytest: 集成测试** - 仅运行集成测试

#### tasks.json - 任务配置

提供7种任务：
1. **运行当前文件测试** - 快速测试当前文件
2. **运行当前目录测试** - 测试整个目录
3. **运行全部测试** - 完整测试套件
4. **运行单元测试** - 仅单元测试
5. **运行集成测试** - 仅集成测试
6. **生成覆盖率报告** - 生成详细覆盖率报告
7. **清理测试报告** - 清理旧报告

#### settings.json - 设置配置

- Python测试框架配置
- 自动测试发现
- 调试端口配置
- Python路径配置
- 代码格式化配置

### 2. pytest配置 (pytest.ini)

```ini
[pytest]
testpaths = 测试
python_files = test_*.py
python_classes = Test*
python_functions = test_*
addopts = -v --tb=short --strict-markers --disable-warnings --color=yes
markers =
    unit: 单元测试
    integration: 集成测试
    slow: 慢速测试
    gui: GUI测试
    api: API测试
```

### 3. 智能联动脚本

#### 测试联动工具.py

功能：
- 自动识别选中文件的测试模块
- 智能匹配对应的测试用例
- 执行测试并生成报告
- 实时展示测试结果

使用方式：
```bash

# 运行全部测试
python AI产物/工具/脚本/测试联动工具.py --all --coverage
```

#### 智能测试运行器.py

功能：
- 监听文件变化自动运行测试
- 智能识别修改的模块
- 增量测试执行
- 实时结果反馈

使用方式：
```bash
# 监听模式
python AI产物/工具/脚本/智能测试运行器.py --watch

# 运行单个文件
python AI产物/工具/脚本/智能测试运行器.py --file <文件路径>
```

## 使用指南

### 方式1：使用VS Code界面

1. **打开测试文件**
   - 在资源管理器中选中测试文件

2. **选择调试配置**
   - 按 `Ctrl+Shift+D` 打开调试面板
   - 选择 "pytest: 当前文件" 或其他配置

3. **运行测试**
   - 按 `F5` 或点击运行按钮
   - 在终端查看测试结果

### 方式2：使用命令面板

1. **打开命令面板**
   - 按 `Ctrl+Shift+P`

2. **运行任务**
   - 输入 "任务: 运行任务"
   - 选择相应的测试任务

### 方式3：使用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+T` | 运行当前文件测试 |
| `Ctrl+Shift+R` | 运行相关测试 |
| `Ctrl+Shift+A` | 运行全部测试 |
| `Ctrl+Shift+U` | 运行单元测试 |
| `Ctrl+Shift+I` | 运行集成测试 |

### 方式4：使用终端命令

```bash
# 运行全部测试
pytest 测试 -v

# 运行单元测试
pytest 测试/单元测试 -v

# 运行带覆盖率的测试
pytest 测试 -v --cov=前端 --cov-report=html

# 运行特定标记的测试
pytest 测试 -v -m unit

# 运行特定文件
pytest 测试/单元测试/test_配置服务.py -v
```
