| 转换数位格式 | 数字固定位数格式化 | `str(num).zfill(width)` |

**Python实现示例**:
```python
import random
import time

def random_delay(base_ms: int, multiplier: float = 1.0):
    min_delay = int(base_ms * multiplier)
    max_delay = int(base_ms * (multiplier + 0.2))
    actual_delay = random.randint(min_delay, max_delay) / 1000
    time.sleep(actual_delay)
```

### 3.3 图色识别模块

| 函数名 | 功能描述 | Python实现方案 |
|--------|---------|---------------|
| 精查单色标识 | 精确查找单色点 | `pyautogui.locateOnScreen()` / OpenCV |
| 查询单色标识 | 多相似度查找颜色 | OpenCV + 多阈值匹配 |
| 查询指定色数 | 统计区域颜色数量 | OpenCV `cv2.inRange()` |

**Python实现方案**:
```python
import cv2
import numpy as np
import pyautogui

class ColorDetector:
    @staticmethod
    def find_color(x1, y1, x2, y2, color_hex: str, similarity: float = 1.0):
        screenshot = pyautogui.screenshot(region=(x1, y1, x2-x1, y2-y1))
        img = cv2.cvtColor(np.array(screenshot), cv2.COLOR_RGB2BGR)
        target_color = tuple(int(color_hex[i:i+2], 16) for i in (0, 2, 4))
        mask = cv2.inRange(img, 
            np.array([c * (1 - similarity) for c in target_color]),
            np.array([255 - (255 - c) * (1 - similarity) for c in target_color]))
        positions = cv2.findNonZero(mask)
        return positions[0][0] if positions is not None else None
```

### 3.4 文字识别模块

| 函数名 | 功能描述 | Python实现方案 |
|--------|---------|---------------|
| 识别指定文字 | OCR识别指定文字 | `pytesseract` / `easyocr` / `paddleocr` |
| 识别首行文字 | OCR识别首行文本 | 同上 + 文本定位 |
| 识别全部文字 | OCR识别全部文本 | 同上 |

**Python实现方案**:
```python
import pytesseract
from PIL import Image
import pyautogui

class OCRDetector:
    @staticmethod
    def recognize_text(x1, y1, x2, y2, lang='chi_sim'):
        screenshot = pyautogui.screenshot(region=(x1, y1, x2-x1, y2-y1))
        text = pytesseract.image_to_string(screenshot, lang=lang)
        return text.strip()
    
    @staticmethod
    def find_text(x1, y1, x2, y2, target_text: str, confidence: float = 0.7):
        screenshot = pyautogui.screenshot(region=(x1, y1, x2-x1, y2-y1))
        data = pytesseract.image_to_data(screenshot, lang='chi_sim', output_type=pytesseract.Output.DICT)
        for i, text in enumerate(data['text']):
            if target_text in text and data['conf'][i] >= confidence * 100:
                return {'x': data['left'][i], 'y': data['top'][i], 'text': text}
        return None
```

```python
import pyautogui
import subprocess

class UIController:
    @staticmethod
    def click(x: int, y: int, use_adb: bool = False):
        if use_adb:
            subprocess.run(['adb', 'shell', 'input', 'tap', str(x), str(y)])
        else:
            pyautogui.click(x, y)
    
    @staticmethod
    def swipe(x1, y1, x2, y2, duration: float = 0.5, use_adb: bool = False):
        if use_adb:
            subprocess.run(['adb', 'shell', 'input', 'swipe', 
                          str(x1), str(y1), str(x2), str(y2), str(int(duration * 1000))])
        else:
            pyautogui.drag(x2 - x1, y2 - y1, duration)
```

### 3.6 数据管理模块

| 函数名 | 功能描述 | Python实现方案 |
|--------|---------|---------------|
| 设置共享数据 | 线程间数据共享 | `threading.local()` / 全局字典 |
| 获取共享数据 | 获取共享数据 | 同上 |
| 文件读写 | 配置文件操作 | `json.load()` / `json.dump()` |

**Python实现方案**:
```python
import json
import threading
from typing import Any, Dict

class SharedDataManager:
    _instance = None
    _lock = threading.Lock()
    _data: Dict[str, Any] = {}
    
    def __new__(cls):
        if cls._instance is None:
            with cls._lock:
                if cls._instance is None:
                    cls._instance = super().__new__(cls)
        return cls._instance
    
    def set(self, key: str, value: Any):
        with self._lock:
            self._data[key] = value
    
    def get(self, key: str, default=None):
        with self._lock:
            return self._data.get(key, default)

class ConfigManager:
    @staticmethod
    def load_config(file_path: str) -> dict:
        with open(file_path, 'r', encoding='utf-8') as f:
            return json.load(f)
    
    @staticmethod
    def save_config(file_path: str, config: dict):
        with open(file_path, 'w', encoding='utf-8') as f:
            json.dump(config, f, ensure_ascii=False, indent=2)
```

### 3.7 异常监控模块

| 函数名 | 功能描述 | Python实现方案 |
|--------|---------|---------------|
| 监测卡屏色数 | 检测屏幕变化 | 截图对比 / 颜色统计 |
| 监测处理弹窗 | 检测并处理弹窗 | OCR + 关键词匹配 |
| 异常监控系统 | 后台监控线程 | `threading.Thread` |

**Python实现方案**:
```python
import threading
import time
import cv2
import numpy as np
import pyautogui

class MonitorSystem:
    def __init__(self, check_interval: float = 2.0):
        self.check_interval = check_interval
        self.running = False
        self.monitor_thread = None
        self.last_screen_hash = None
        self.stuck_count = 0
        self.max_stuck_count = 10
    
    def start(self):
        self.running = True
        self.monitor_thread = threading.Thread(target=self._monitor_loop, daemon=True)
        self.monitor_thread.start()
    
    def stop(self):
        self.running = False
        if self.monitor_thread:
            self.monitor_thread.join()
    
    def _monitor_loop(self):
        while self.running:
            current_hash = self._get_screen_hash()
            if current_hash == self.last_screen_hash:
                self.stuck_count += 1
                if self.stuck_count >= self.max_stuck_count:
                    self._handle_stuck()
            else:
                self.stuck_count = 0
            self.last_screen_hash = current_hash
            time.sleep(self.check_interval)
    
    def _get_screen_hash(self):
        screenshot = pyautogui.screenshot()
        img = cv2.cvtColor(np.array(screenshot), cv2.COLOR_RGB2GRAY)
        return hash(img.tobytes())
    
    def _handle_stuck(self):
        # 触发重启逻辑
        pass
```
