## 🎯 **分步构建指南**

### **第一阶段：基础环境验证**
```bash
# 1. 验证Python
python --version

# 2. 安装Flet
pip install flet==0.82.0

# 3. 验证Flet
python -c "import flet; print(f'Flet版本: {flet.__version__}')"
```

### **第二阶段：安装Flutter SDK**
1. **下载Flutter SDK**：https://flutter.dev/docs/get-started/install
2. **解压到合适位置**（如：`C:\flutter`）
3. **添加环境变量**：
   - 系统变量 `Path` 添加：`C:\flutter\bin`
4. **验证安装**：
   ```bash
   flutter --version
   flutter doctor
   ```

### **第三阶段：安装Android SDK**
1. **下载Android Studio**：https://developer.android.com/studio
2. **安装时选择Android SDK组件**
3. **设置环境变量**：
   - `ANDROID_HOME` = `C:\Users\你的用户名\AppData\Local\Android\Sdk`
4. **接受许可**：
   ```bash
   flutter doctor --android-licenses
   ```

### **第四阶段：构建APK**
```bash
# 完整构建
python -m flet.cli build apk --project "WW2Assistant" --verbose

# 或使用脚本
python local_apk_builder.py
```

## 💡 **常见问题解决**

### **问题1：Flutter安装失败**
**解决方案**：
1. 使用国内镜像：https://flutter.cn/community/china
2. 设置环境变量：
   ```bash
   setx PUB_HOSTED_URL https://pub.flutter-io.cn
   setx FLUTTER_STORAGE_BASE_URL https://storage.flutter-io.cn
   ```

### **问题2：Android SDK许可证**
**解决方案**：
```bash
flutter doctor --android-licenses
# 按y接受所有许可
```

### **问题3：构建失败**
**解决方案**：
1. 检查错误信息
2. 确保所有依赖已安装
3. 运行 `flutter doctor` 诊断问题
