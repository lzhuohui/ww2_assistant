# 技能卡片集

## 技能卡片说明

每个技能卡片包含以下信息：
- **技能名称**：技能的唯一标识
- **技能类型**：参考技能、验证技能、脚手架技能、审核技能、工作流技能、调查技能
- **所属层级**：基础层、应用层、高级层
- **触发条件**：技能的触发场景
- **核心功能**：技能的主要功能
- **执行步骤**：技能的执行流程
- **应用场景**：典型应用场景
- **常见问题**：常见问题及解决策略
- **前置技能**：使用该技能前需要掌握的技能
- **依赖技能**：该技能依赖的其他技能

| 属性 | 内容 |
|------|------|
| **技能名称** | 规则查询助手 |
| **技能类型** | 参考技能 |
| **所属层级** | 基础层-规则理解 |
| **触发条件** | 用户询问规则内容、规范要求或需要了解特定规则时触发 |
