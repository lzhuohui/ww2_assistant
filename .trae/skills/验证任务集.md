### 任务3：简单代码审查
**任务描述**：
审查以下代码片段，识别违规问题并提供改进建议：

```python
import flet as ft

# ❌ 违规1：缺少用户指定变量区域

class UserCard:
    """用户卡片组件"""
    
    # ❌ 违规2：函数参数缺少类型注解和默认值
    def create(name, age):
        return ft.Container(
            content=ft.Column([
                ft.Text(name, size=16),
                ft.Text(f"年龄: {age}")
            ])
        )

if __name__ == "__main__":
    # ❌ 违规3：调试逻辑不符合标准（使用了自定义main函数）
    # ❌ 违规4：在调试逻辑中设置页面属性
    # ❌ 违规5：使用非默认参数调用
    def main(page):
        page.title = "用户卡片"
        card = UserCard.create("张三", 25)
        page.add(card)
    ft.run(main)
```

**预期识别的违规点**：
1. 缺少模块级文档字符串
2. 缺少用户指定变量区域
3. 函数参数缺少类型注解和默认值
4. 调试逻辑不符合标准格式
5. 在调试逻辑中设置页面属性
6. 使用非默认参数调用模块

**验证目标**：
- 代码审查助手技能
- 规范合规检查器技能

**验收标准**：
- [ ] 识别出缺少文档字符串
- [ ] 识别出缺少用户变量区域
- [ ] 识别出函数参数缺少默认值和类型注解
- [ ] 识别出调试逻辑不符合标准
- [ ] 提供具体的改进建议和代码示例

**评分标准**：
- 问题识别：50分（识别出所有违规问题）
- 建议质量：30分（提供具体可行的改进建议）
- 示例代码：20分（提供正确的代码示例）

**参考正确实现**：
```python
# -*- coding: utf-8 -*-
"""模块名称：UserCard | 用户卡片展示组件 | 模块隔离原则"""
import flet as ft

# *** 用户指定变量 - AI不得修改 ***
# （用户未指定变量）
# *********************************

class UserCard:
    """用户卡片组件，用于展示用户信息"""
    
    @staticmethod
    def create(name: str = "默认名称", age: int = 18) -> ft.Container:
        """创建用户卡片
        
        Args:
            name: 用户姓名
            age: 用户年龄
        
        Returns:
            ft.Container: 用户卡片容器
        """
        return ft.Container(
            content=ft.Column([
                ft.Text(name, size=16),
                ft.Text(f"年龄: {age}")
            ])
        )

if __name__ == "__main__":
    ft.run(lambda page: page.add(UserCard.create()))
```
