# 6A流程技能总目录

## 概述
本目录包含完整的6A开发流程技能，用于规范化手游辅助项目的开发流程。

## 6A流程说明
6A流程是一个系统化的AI应用开发方法论，包含六个阶段：

1. **Align（需求对齐）** - 澄清需求，定义验收标准
2. **Architect（技术架构）** - 设计系统分层、模块划分和接口定义
3. **Atomize（原子化拆分）** - 将复杂任务拆分为1-8小时的原子任务
4. **Approve（风险审批）** - 检查数据合规、模型可行性
5. **Automate（自动化实施）** - 生成代码模板、实时检查规范
6. **Assess（交付评估）** - 汇总测试结果，生成验收报告

## 使用方式

### 启动完整6A流程
```
@6A 新任务
```
或
```
开始6A流程
```

### 单独使用某个阶段技能
```
@6a-architect 设计架构
```

## 输出文档
每个阶段会生成对应的文档：
- `ALIGNMENT_任务名.md` - 需求对齐文档
- `DESIGN_任务名.md` - 技术架构文档
- `TASKS_任务名.md` - 任务拆分文档
- `APPROVAL_任务名.md` - 风险审批文档
- `ACCEPTANCE_任务名.md` - 交付评估文档

## 特点
- 规范化开发流程
- 自动生成文档
