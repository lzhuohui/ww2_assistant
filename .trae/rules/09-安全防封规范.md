# 安全与防封规范

## 操作随机化（必须）
- **坐标偏移**：所有点击/滑动坐标添加±5像素随机偏移
- **时间间隔**：基础时间±30%随机化

```python
def safe_tap(adb, x, y):
    x += random.randint(-5,5); y += random.randint(-5,5)
    adb.tap(x, y)
    time.sleep(random.uniform(1.0, 2.5))
```

## 行为模式
- 避免长时间无操作（每10-15分钟随机切屏）
- 游戏异常退出需有冷静期（暂停30-60秒）
- 每日运行时长≤12小时

## 敏感信息保护
- 禁止硬编码API密钥、账号密码
- 使用环境变量或加密配置文件
- 配置文件模板上传仓库，实际文件加入`.gitignore`

## 检测规避
- 随机化技能释放顺序
- 模拟真实设备行为（偶尔停顿）
- 避免固定脚本特征
