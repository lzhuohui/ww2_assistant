# 开发流程 - 模块调用（导入和调用）

## 导入规范

**正确导入方式：**
```python
# ✅ 正确：从项目内部模块导入
from 前端.用户设置界面.单元模块.文本标签 import LabelText
from 前端.用户设置界面.单元模块.通用按钮 import Button

# ❌ 错误：直接使用flet基础控件，未复用封装模块
import flet as ft
# 然后直接使用 ft.Text, ft.Button 等
```

**导入顺序：**
```python
# 1. 标准库
import sys
from pathlib import Path

# 2. 第三方库
import flet as ft

# 3. 项目内部模块（按层级排序）
from 前端.用户设置界面.单元模块.文本标签 import LabelText
from 前端.用户设置界面.单元模块.通用按钮 import Button
```

## 调用规范

**通过公开接口调用：**
```python
# ✅ 正确：通过create()公开接口调用
title = LabelText.create(text="标题", role="h2")
button = Button.create(text="点击", style="text", width=150)

# ❌ 错误：直接访问内部实现或私有属性
button._internal_state = "xxx"  # 访问私有属性
```

**参数传递规范：**
```python
# ✅ 正确：通过参数传递调整行为
title = LabelText.create(text="自定义标题", role="h1", size=32)
button = Button.create(text="自定义按钮", style="nav", width=200, height=50)
```
