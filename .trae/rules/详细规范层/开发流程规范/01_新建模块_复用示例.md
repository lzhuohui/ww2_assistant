# 开发流程 - 新建模块（复用示例）

## 步骤3：优先复用现有模块

**在模块内部实现时：**

```python
# ✅ 正确：在新建模块中复用现有模块
def create():
    from 前端.用户设置界面.单元模块.文本标签 import LabelText
    from 前端.用户设置界面.单元模块.通用按钮 import Button
    
    title = LabelText.create(text="标题", role="h2")  # 复用现有模块
    button = Button.create(text="点击", style="text")  # 复用现有模块
    
    return ft.Container(content=ft.Column([title, button]))

# ❌ 错误：直接创建基础控件，未复用现有模块
def create():
    title = ft.Text("标题", size=24)  # 应该使用LabelText
    button = ft.Button("点击")  # 应该使用Button
    return ft.Container(content=ft.Column([title, button]))
```
