# 开发流程 - 新建模块（模块结构）

## 步骤2：创建模块结构

按照【代码结构规范】创建完整模块：

```python
# -*- coding: utf-8 -*-
"""
模块名称：XXX
设计思路及联动逻辑: 说明为什么需要新建此模块（必须说明无法复用的原因）
模块隔离原则: 1. 不直接创建被调用模块的内容 2. 不覆盖被调用模块的计算结果 3. 用户指定变量除外
"""

import flet as ft

# *** 用户指定变量 - AI不得修改, 变量值必须生效 ***
# （用户未指定变量）
# *********************************

class XXX:
    """XXX - 简要描述"""
    
    @staticmethod
    def create(...) -> ft.Container:
        # 优先调用现有模块，而非直接创建控件
        return ft.Container()

# *** 调试逻辑 ***
if __name__ == "__main__":
    ft.run(lambda page: page.add(XXX.create()))
```
