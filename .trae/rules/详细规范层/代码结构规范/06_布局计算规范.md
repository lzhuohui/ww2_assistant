# 布局计算规范

## 适用场景
- 创建自适应布局时
- 计算组件尺寸时
- 计算组件位置时

## 核心原则

### 自适应布局
- 根据组件的实际尺寸和位置进行布局计算
- 避免硬编码尺寸和位置
- 使用动态计算

### 尺寸获取
- 从子组件中获取实际尺寸信息
- 使用 `hasattr` 检查属性存在
- 提供默认值避免错误

### 位置计算
- 基于基准元素（如分割线）计算其他元素的位置
- 使用相对位置而非绝对位置
- 考虑边距和内边距

## 标准实践

```python
# 获取容器宽度
title_container_width = title_container_content.width

# 获取分割线宽度（带默认值）
actual_divider_width = divider.width if hasattr(divider, 'width') and divider.width is not None else 100

# 计算相对位置
left_position = divider.left + divider.width + padding
```

## 注意事项

- 避免硬编码尺寸
- 使用动态计算
- 提供默认值
- 检查属性存在性
