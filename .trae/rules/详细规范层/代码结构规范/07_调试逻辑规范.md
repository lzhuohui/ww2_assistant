# 调试逻辑规范

## 标准格式（强制）

```python
# *** 调试逻辑 ***
if __name__ == "__main__":
    ft.run(lambda page: page.add(XXX.create()))
```

## 严格禁止

1. **禁止包装函数**：勿使用`def main(page):`形式包装
2. **禁止页面配置**：勿设置`page.title`,`page.padding`,`page.bgcolor`
3. **禁止硬编码参数**：勿传入非默认参数
4. **禁止回调包装**：勿创建额外的回调函数包装
5. **禁止初始化代码**：勿在调试逻辑中添加初始化代码
6. **只能替换模块名**：`XXX`只能替换为被测模块名称

## 违规示例

```python
# ❌ 错误：使用包装函数
def main(page: ft.Page):
    page.title = "XXX"
    page.add(XXX.create())
ft.run(main)

# ❌ 错误：硬编码参数
ft.run(lambda page: page.add(XXX.create(title="自定义")))

# ❌ 错误：在调试逻辑中添加初始化代码
配置 = 界面配置()
ThemeProvider.initialize(配置)
```

## 正确示例

```python
# ✅ 正确：标准格式
if __name__ == "__main__":
    ft.run(lambda page: page.add(XXX.create()))
```

## 依赖初始化处理

如果被测模块依赖外部初始化，**必须在`create`方法内部处理**：

```python
@staticmethod
def create(...) -> ft.Container:
    if ThemeProvider._config is None:
        ThemeProvider.initialize(界面配置())
    ...
```
