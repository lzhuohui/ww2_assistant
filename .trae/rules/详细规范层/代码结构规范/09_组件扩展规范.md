# 组件扩展规范

## 适用场景
- 需要扩展组件功能时
- 需要添加状态管理时
- 需要添加属性访问时

## 核心原则

### 功能扩展
- 通过向返回的容器对象添加方法
- 扩展组件的功能
- 不修改原始组件

### 状态管理
- 提供状态设置、切换和获取的方法
- 封装状态管理逻辑
- 保持接口一致性

### 属性访问
- 允许外部代码访问和修改组件的属性
- 提供getter和setter方法
- 保持封装性

## 标准实践

```python
# 功能扩展
container.set_state = set_state
container.toggle_state = toggle_state
container.get_state = get_state
container.set_subtitle = set_subtitle

# 状态管理
def set_state(enabled: bool):
    container.opacity = 1.0 if enabled else 0.4
    if container.page:
        container.update()

def toggle_state():
    current = container.get_state()
    set_state(not current)

def get_state() -> bool:
    return container.opacity == 1.0

# 属性访问
def set_subtitle(text: str):
    subtitle_text.value = text
    if subtitle_text.page:
        subtitle_text.update()
```

## 注意事项

- 不修改原始组件
- 保持接口一致性
- 封装内部实现
- 提供清晰的API
