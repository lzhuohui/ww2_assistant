# 函数参数规范

## 参数默认值（强制）

```python
def create(
    title: str="测试标题",
    icon: str="HOME",
    enabled: bool=True,
) -> ft.Container:
```

## 默认值要求

- 所有参数必须有默认值
- **默认值必须能展示完整功能效果**

## 禁止的默认值

| 违规示例 | 问题 | 正确示例 |
|----------|------|----------|
| `icon: str=None` | 不显示 | `icon: str="SETTINGS"` |
| `icon: str=""` | 不显示 | `icon: str="SETTINGS"` |
| `items: list=[]` | 不显示 | `items: list=["A", "B"]` |

## 参数格式（PEP 8）

`=`两边不加空格：
```python
def create(title: str="默认值", size: int=16):
    pass
```
