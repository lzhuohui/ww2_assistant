# 规范管理 - 配置优化（配置原则）

### 3.2 规则配置优化方案

#### 配置原则

1. **核心规范**：alwaysApply: true，确保基本规范始终生效
2. **详细规范**：alwaysApply: false，description明确适用场景
3. **检查清单**：alwaysApply: false，description明确使用时机
4. **测试用例**：不作为规则，不配置alwaysApply

#### 配置示例

**核心规范配置：**
```yaml
---
alwaysApply: true
globs: ["**/*.py"]
---
# 代码结构规范（精简版）

## 核心原则
...
```

**详细规范配置：**
```yaml
---
alwaysApply: false
description: "创建新模块或重构模块结构时使用，确保模块符合标准结构要求"
---
# 模块结构规范

## 适用场景
...
```

**检查清单配置：**
```yaml
---
alwaysApply: false
description: "代码审查时使用，系统性检查代码是否符合所有规范要求"
---
# 代码结构规范检查清单

## 使用说明
...
```
