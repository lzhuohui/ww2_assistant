# 用户指定变量规范

## 标准格式

```python
# *** 用户指定变量 - AI不得修改, 变量值必须生效 ***
USER_XXX = 16  # 中文说明
# *********************************
```

无用户指定变量时：
```python
# *** 用户指定变量 - AI不得修改, 变量值必须生效 ***
# （用户未指定变量）
# *********************************
```

## 命名规范（强制）

| 正确 | 错误 |
|------|------|
| `USER_WIDTH = 240` | ❌ `DEFAULT_WIDTH = 240` |
| `USER_HEIGHT = 34` | ❌ `HEIGHT = 34` |

- **必须**使用`USER_`前缀
- 每个变量必须添加中文注释

## 严格禁止

- ❌ 使用`DEFAULT_`前缀
- ❌ 使用无前缀命名
- ❌ 省略用户指定变量区域
- ❌ 省略中文注释

## AI检查清单

- [ ] 是否使用`USER_`前缀？
- [ ] 是否添加了中文注释？
