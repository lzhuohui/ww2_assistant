# 布局计算规范

## 适用场景
- 创建自适应布局时
- 计算组件尺寸时
- 计算组件位置时

## 核心原则

### 自适应布局
- 根据组件的实际尺寸和位置进行布局计算
- 避免硬编码尺寸和位置
- 使用动态计算

### 尺寸获取方式（明确区分）

| 方式 | 适用场景 | 获取方法 | 示例 |
|------|----------|----------|------|
| **静态尺寸** | 固定配置值 | 从配置模块获取 | `config.获取尺寸("图标标题", "container_width")` |
| **动态尺寸** | 运行时计算 | 从组件属性获取 | `component.width` |
| **用户指定** | 用户可调参数 | 从用户变量区域 | `USER_CONTAINER_WIDTH` |

#### 静态尺寸获取
```python
# 从配置获取固定尺寸
container_width = config.获取尺寸("图标标题", "container_width")
line_width = config.获取尺寸("图标标题", "divider_width")
```

#### 动态尺寸获取
```python
# 从组件获取实际尺寸（使用hasattr检查）
actual_width = component.width if hasattr(component, 'width') and component.width is not None else 100

# 获取容器内容尺寸
title_container_width = title_container_content.width
title_container_height = title_container_content.height
```

#### 用户指定尺寸
```python
# 直接使用用户指定变量
width = USER_CONTAINER_WIDTH
height = USER_CONTAINER_HEIGHT
```

### 位置计算
- 基于基准元素（如分割线）计算其他元素的位置
- 使用相对位置而非绝对位置
- 考虑边距和内边距

## 标准实践

```python
# 1. 获取静态配置尺寸
container_width = config.获取尺寸("图标标题", "container_width")

# 2. 获取动态组件尺寸
title_container_width = title_container_content.width

# 3. 计算相对位置
left_position = divider.left + divider.width + padding
```

## 注意事项

- 避免硬编码尺寸
- 使用动态计算
- 提供默认值
- 检查属性存在性
