# 函数参数规范

## 参数默认值（强制）

```python
def create(
    title: str="测试标题",
    icon: str="HOME",
    enabled: bool=True,
) -> ft.Container:
```

## 默认值要求

- 所有参数必须有默认值
- **默认值必须能展示完整功能效果**

## 默认值来源优先级

| 优先级 | 来源 | 使用场景 | 示例 |
|--------|------|----------|------|
| 1 | **用户指定变量** | 用户明确指定了变量 | `width: int = USER_WIDTH` |
| 2 | **硬编码默认值** | 用户未指定，使用合理默认值 | `enabled: bool = True` |
| 3 | **配置获取** | 需要从配置文件读取 | `theme: str = config.get_theme()` |

### 用户指定变量引用方式
```python
# *** 用户指定变量 - AI不得修改, 变量值必须生效 ***
USER_WIDTH = 20  # 容器宽度
USER_HEIGHT = 100  # 容器高度
# *********************************

def create(
    width: int = USER_WIDTH,   # 正确：引用用户指定变量
    height: int = USER_HEIGHT, # 正确：引用用户指定变量
    enabled: bool = True       # 正确：用户未指定，使用硬编码
) -> ft.Container:
    pass
```

## 禁止的默认值

| 违规示例 | 问题 | 正确示例 |
|----------|------|----------|
| `icon: str=None` | 不显示 | `icon: str="SETTINGS"` |
| `icon: str=""` | 不显示 | `icon: str="SETTINGS"` |
| `items: list=[]` | 不显示 | `items: list=["A", "B"]` |

## 参数格式（PEP 8）

`=`两边不加空格：
```python
def create(title: str="默认值", size: int=16):
    pass
```
