# 调试逻辑规范

## 标准格式（强制）

```python
# *** 调试逻辑 ***
if __name__ == "__main__":
    ft.run(lambda page: page.add(XXX.create()))
```

## 严格禁止

1. ❌ 禁止包装函数：`def main(page):`
2. ❌ 禁止页面配置：`page.title`, `page.padding`
3. ❌ 禁止硬编码具体值：传入具体值（如`title="测试"`）
4. ❌ 禁止初始化代码
5. ❌ 只能替换模块名：`XXX`替换为被测模块名

## 允许的情况

✅ **允许引用用户指定变量**：
```python
# 正确：使用用户指定变量（非硬编码具体值）
ft.run(lambda page: page.add(Divider.create(width=USER_CONTAINER_WIDTH)))
```

## 违规示例

```python
# ❌ 错误：硬编码具体值
ft.run(lambda page: page.add(XXX.create(title="测试")))

# ❌ 错误：硬编码数值
ft.run(lambda page: page.add(Divider.create(width=100)))

# ❌ 错误：初始化代码
配置 = 界面配置()
ft.run(lambda page: page.add(XXX.create()))
```
