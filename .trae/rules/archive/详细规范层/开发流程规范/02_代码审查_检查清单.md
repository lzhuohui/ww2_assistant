# 开发流程 - 代码审查（检查清单）

## 审查检查清单

### 模块复用检查（新增）
- [ ] 已检查是否搜索现有模块
- [ ] 已检查是否存在功能重复
- [ ] 已检查是否正确复用
- [ ] 已记录复用审查结果

### 模块结构检查
- [ ] 文件头正确
- [ ] 文档字符串完整
- [ ] 导入顺序正确
- [ ] 用户指定变量区域正确
- [ ] 类/函数定义符合规范
- [ ] 调试逻辑标准

### 模块隔离检查
- [ ] 调用者遵守隔离原则
- [ ] 被调模块接口清晰
- [ ] 数据传递正确

### 代码质量检查
- [ ] 函数长度合规
- [ ] 命名规范
- [ ] 无冗余代码

### 测试验证
- [ ] 功能正确
- [ ] 无错误

## 执行口诀

> **"先查复用，再看结构；**  
> **隔离原则，质量把关；**  
> **重复功能，绝不放过"**
