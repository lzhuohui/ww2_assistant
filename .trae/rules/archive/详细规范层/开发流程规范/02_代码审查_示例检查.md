# 开发流程 - 代码审查（示例和检查清单）

## 审查示例

### 场景1：发现未复用现有模块

**审查代码：**
```python
def create():
    # ❌ 问题：直接创建Text，未使用LabelText
    title = ft.Text("标题", size=24, weight=ft.FontWeight.BOLD)
    return title
```

**审查结果：**
```
模块复用审查结果：
- 已复用模块：无
- 未复用但应复用的模块：文本标签.py (LabelText)
- 建议：使用 LabelText.create(text="标题", role="h2") 替换 ft.Text
```

### 场景2：发现功能重复

**审查代码：**
```python
# 新建模块：我的按钮.py
class MyButton:
    def create(text="按钮"):
        return ft.Container(content=ft.Text(text))
```

**现有模块：**
```python
# 已存在：通用按钮.py
class Button:
    def create(text="按钮", style="text", ...):
        ...
```

**审查结果：**
```
模块复用审查结果：
- 已复用模块：无
- 未复用但应复用的模块：通用按钮.py (Button)
- 问题：功能与现有模块重复
- 建议：删除我的按钮.py，使用通用按钮.py并通过参数调整
- 判定：严重违规
```
