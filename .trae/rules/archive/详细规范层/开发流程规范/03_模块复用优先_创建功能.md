# 开发流程 - 模块复用优先原则（创建新功能）

### 2.1 创建新功能时

**执行流程：**
```
1. 搜索现有模块 → 2. 评估复用性 → 3. 决策（复用/创建）
```

**禁止行为：**
- ❌ 未搜索就直接创建新模块
- ❌ 已有相似模块却重新实现
- ❌ 复制现有模块代码修改后使用

**正确示例：**
```python
# ✅ 正确：复用现有文本标签和按钮模块
from 前端.用户设置界面.单元模块.文本标签 import LabelText
from 前端.用户设置界面.单元模块.通用按钮 import Button

def create_component():
    title = LabelText.create(text="标题", role="h2")
    button = Button.create(text="点击", style="text")
    return ft.Container(content=ft.Column([title, button]))
```
