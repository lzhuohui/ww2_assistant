# 开发流程 - 模块复用优先原则（搜索方法）

## 三、模块搜索方法

### 3.1 搜索路径优先级

```
1. 项目内单元模块（前端/用户设置界面/单元模块/）
2. 项目内组件模块（前端/用户设置界面/组件模块/）
3. 项目内工具模块（前端/用户设置界面/工具/）
4. 项目内核心接口（前端/用户设置界面/核心接口/）
5. 项目内配置模块（前端/用户设置界面/配置/）
6. 第三方库（flet等）
```

### 3.2 搜索关键词

| 需求类型 | 搜索关键词 |
|----------|-----------|
| 文本显示 | 文本标签、LabelText、Text |
| 按钮 | 通用按钮、Button、按钮 |
| 容器 | 通用容器、卡片容器、Container |
| 输入 | 输入框、下拉框、输入验证 |
| 图标 | 容器图标、用户头像、Icon |
| 布局 | 分割线、功能容器、导航按钮 |

### 3.3 快速核查清单

**创建/修改前必查：**
- [ ] 已搜索项目中所有相关模块
- [ ] 已阅读匹配模块的文档字符串
- [ ] 已确认模块的公开接口
- [ ] 已评估参数配置可行性
- [ ] 已决策复用方式（直接/参数调整/组合）
