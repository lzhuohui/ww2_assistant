# 检查清单 - 用户变量

## 用户指定变量规范检查

### 区域检查
- [ ] 区域必须存在（不可省略）
- [ ] 使用标准格式的隔离区域
- [ ] 用户意图判断正确：
  - 用户明确指定变量 → 定义具体变量
  - 用户未指定变量 → 标注 `# （用户未指定变量）`

### 变量检查（如有变量）
- [ ] 使用`USER_`前缀
- [ ] 变量值在代码中实际生效
- [ ] 未被二次覆盖
- [ ] 命名具有描述性
- [ ] 注释说明变量用途

### 禁止项检查
- [ ] 未将函数默认参数放在用户指定变量区域
- [ ] 未使用 `DEFAULT_` 前缀的模块级变量
- [ ] 用户未指定时未创建 `USER_` 变量

## 导入规范检查

### 导入顺序
- [ ] 导入顺序正确（标准库 → 第三方库 → 项目内部）
- [ ] 每组内按字母顺序排序
- [ ] 无未使用的导入
- [ ] 无通配符导入
- [ ] 无循环导入
- [ ] 格式符合PEP 8规范

## 函数参数规范检查

### 默认值检查
- [ ] 所有参数都有默认值（除特殊情况）
- [ ] 默认值能展示完整功能效果
- [ ] 默认值来源正确：
  - 用户未指定 → 直接硬编码默认值
  - 用户明确指定 → 引用用户指定变量
  - 通用配置 → 从配置文件获取

### 格式检查
- [ ] 参数传递时`=`两边不加空格
- [ ] 参数命名使用snake_case
- [ ] 参数具有描述性名称
- [ ] 所有参数都有类型注解
- [ ] 返回值有类型注解
