# 检查清单

## 调用者检查清单
- [ ] 是否只通过公开接口访问模块？
- [ ] 是否避免访问私有属性（`_`前缀）？
- [ ] 是否避免修改被调模块的内部数据？
- [ ] 是否通过接口修改被调模块的状态？
- [ ] 是否避免依赖被调模块的实现细节？
- [ ] 是否避免直接修改被调模块的用户指定变量？
- [ ] 是否通过参数传递覆盖被调模块的默认值？

## 被调模块检查清单
- [ ] 是否明确标记公开接口？
- [ ] 是否将内部实现标记为私有？
- [ ] 是否返回数据副本而非内部引用？
- [ ] 是否提供了清晰的接口文档？
- [ ] 是否隐藏了实现细节？
- [ ] 是否正确设置用户指定变量区域？
- [ ] 是否保护用户指定变量不被修改？
