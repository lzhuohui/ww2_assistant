# 用户指定变量规范

## 标准格式
```python
# *** 用户指定变量 - AI不得修改 ***
DEFAULT_WIDTH = 120  # 默认宽度
DEFAULT_HEIGHT = 80  # 默认高度
# *********************************
```

## 核心规则
- **禁止修改**：AI不得修改、删除或添加用户指定变量区域的任何内容
- **禁止覆盖**：不得在同一模块内二次覆盖用户指定变量
- **优先级**：调用方 > 被调用方 > 配置文件
- **隔离保护**：调用方不得直接修改被调用方的用户指定变量

## 实现机制
1. **严格隔离**：用户指定变量区域与代码实现严格分离
2. **明确标记**：使用固定格式的分隔线标记变量区域
3. **只读保护**：AI在处理过程中必须保持该区域完全不变
4. **继承机制**：调用方可以通过参数传递覆盖被调用方的默认值

## 违反处理
- ⚠️ 轻微修改：警告并恢复原始内容
- 🔴 严重修改：拒绝执行并提示重新输入
- 🚫 多次违反：记录并限制功能
