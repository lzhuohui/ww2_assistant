# 下拉框测试目录

这是一个完全独立的下拉框测试环境，用于验证懒加载下拉框的功能和性能。

## 目录结构

```
下拉框测试/
├── README.md                    # 本说明文件
├── 下拉框_懒加载版.py           # 懒加载下拉框核心组件
├── 下拉框_浮动版.py             # 浮动下拉框组件（不推挤控件）
├── 测试_懒加载下拉框.py         # 完整功能测试脚本
├── 测试_浮动下拉框.py           # 浮动效果测试脚本
├── 对比测试_浮动效果.py         # 传统vs浮动对比测试
└── 简单测试.py                  # 最简化测试脚本
```

## 组件说明

### 下拉框_懒加载版.py
- **功能**: 实现了基于"共识 #020"的懒加载下拉框
- **特点**:
  - 点击时才加载选项，减少内存占用
  - 任何时候只有一个下拉框显示选项列表
  - 关闭下拉框时销毁选项列表
  - 支持自定义宽度、高度、主题颜色
- **核心类**: `LazyDropdownForBuilding`
- **工厂函数**: `create_lazy_dropdown()`

### 下拉框_浮动版.py
- **功能**: 解决了下拉框弹出面板推挤控件的问题
- **特点**:
  - 使用Stack布局实现浮动效果
  - 弹出面板浮在现有控件之上，不会推挤下方控件
  - 保持界面稳定，避免跳动
  - 同样支持懒加载功能
- **核心类**: `FloatingDropdown`
- **工厂函数**: `create_floating_dropdown()`

### 测试_懒加载下拉框.py
- **功能**: 完整的测试脚本，包含多种测试场景
- **测试内容**:
  1. 基础懒加载功能测试
  2. 建筑配置区模拟测试
  3. 性能测试（多个下拉框）
  4. 日志记录和调试功能
- **运行方式**: `python 测试_懒加载下拉框.py`

### 测试_浮动下拉框.py
- **功能**: 专门测试浮动效果的脚本
- **特点**: 验证下拉框不推挤下方控件的效果
- **运行方式**: `python 测试_浮动下拉框.py`

### 对比测试_浮动效果.py
- **功能**: 传统下拉框vs浮动下拉框对比测试
- **特点**: 直观展示浮动效果的优势
- **运行方式**: `python 对比测试_浮动效果.py`

### 简单测试.py
- **功能**: 最简化的测试脚本，用于快速验证
- **特点**: 只包含3个基本下拉框，便于调试
- **运行方式**: `python 简单测试.py`

## 核心特性

### 懒加载原理
基于"共识 #020：下拉框懒加载优化方案"：

1. **点击加载**: 用户点击下拉框时才加载选项数据
2. **单活动**: 任何时候只有一个下拉框显示选项列表
3. **销毁机制**: 关闭下拉框时销毁选项列表，释放内存
4. **性能优化**: 大幅减少初始化时的对象创建数量

### 浮动效果原理
解决下拉框弹出面板推挤控件的问题：

1. **Stack布局**: 使用Flet的Stack组件实现层叠效果
2. **浮动定位**: 弹出面板使用绝对定位，浮在现有控件之上
3. **不推挤**: 弹出面板不会改变其他控件的位置
4. **界面稳定**: 避免界面跳动，提升用户体验
