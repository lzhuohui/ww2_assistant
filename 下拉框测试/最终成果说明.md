## 🚀 如何使用

### 1. 导入组件和数据
```python
from 下拉框_合格版 import create_dropdown
from 建筑选项 import get_city_options, get_building_levels, get_building_types
import json
```

### 2. 加载配置
```python
# 加载JSON配置
with open("建筑配置.json", "r", encoding="utf-8") as f:
    config = json.load(f)

# 获取用户偏好
user_prefs = config.get("用户偏好", {})
current_city = user_prefs.get("最近使用的城市", "北京")
```

### 3. 创建下拉框（使用配置数据）
```python
# 城市选择 - 使用配置中的上次选择
city_dropdown = create_dropdown(
    options=get_city_options(),  # 从建筑选项.py获取
    current_value=current_city,  # 从配置加载
    width=180,
    on_change=lambda v: save_to_config("最近使用的城市", v),
)

# 建筑等级 - 懒加载
level_dropdown = create_dropdown(
    current_value="17",
    width=100,
    option_loader=get_building_levels,  # 懒加载函数
    on_change=lambda v: save_to_config("最近使用的等级", v),
)

# 建筑类型
building_dropdown = create_dropdown(
    options=get_building_types(),
    current_value="市政厅",
    width=180,
    on_change=lambda v: save_to_config("最近使用的建筑", v),
)
```

### 3. 控制方法
```python
# 获取当前值
current_value = dropdown.get_value()

# 设置值
dropdown.set_value("上海")

# 启用/禁用
dropdown.set_enabled(True)  # 启用
dropdown.set_enabled(False) # 禁用
```
