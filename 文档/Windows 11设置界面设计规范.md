# Windows 11 设置界面设计规范

## 一、设计理念

### 1.1 Fluent Design System

Windows 11 采用 Fluent Design System 设计语言，包含五大核心元素：

| 元素 | 英文 | 说明 |
|------|------|------|
| 光感 | Light | 创造良好的氛围，让应用有空间概念 |
| 深度 | Depth | 通过亚克力材料提供细腻的场景转换 |
| 动画 | Motion | 全新的过渡动画、视觉效果 |
| 材质 | Material | 半透明和模糊效果（毛玻璃效果） |
| 缩放 | Scale | 响应式布局适配 |

### 1.2 设计目标

- **平静的技术**：让生活真正变得更好
- **情感联系**：软化冰冷的UI，增加情感联系
- **自然流畅**：应用的使用更加自然和流畅

## 四、交互状态

### 4.1 状态类型

| 状态 | 英文 | 说明 |
|------|------|------|
| 正常 | Normal | 默认状态 |
| 悬停 | Hover | 鼠标悬停状态 |
| 按下 | Pressed | 鼠标按下状态 |
| 禁用 | Disabled | 不可用状态 |
| 聚焦 | Focus | 键盘聚焦状态 |

### 4.2 状态效果

#### 4.2.1 悬停状态（Hover）

**背景色变化**：
- 深色主题：背景色从透明变为 `#3D3D3D`（悬停背景色）
- 浅色主题：背景色从透明变为 `#E5E5E5`（悬停背景色）

**透明度变化**：
- 背景透明度：从 `0` 变为 `0.1`
- 边框透明度：从 `0` 变为 `0.5`

**发光效果（Reveal Effect）**：
- 鼠标位置周围显示光晕效果
- 光晕颜色：强调色（`#0078D4`）
- 光晕半径：`20-40px`
- 光晕透明度：`0.1-0.3`

#### 4.2.2 按下状态（Pressed）

**背景色变化**：
- 深色主题：背景色变为 `#4A4A4A`（比悬停更深）
- 浅色主题：背景色变为 `#D0D0D0`（比悬停更深）

**透明度变化**：
- 背景透明度：`0.15`
- 边框透明度：`0.8`

**缩放效果**：
- 轻微缩放：`scale(0.98)`

#### 4.2.3 禁用状态（Disabled）

**背景色变化**：
- 背景色变为透明或 `#2A2A2A`（深色主题）
- 背景色变为透明或 `#F5F5F5`（浅色主题）

**文字色变化**：
- 文字色变为 `#5C5C5C`（深色主题）
- 文字色变为 `#CCCCCC`（浅色主题）

**透明度变化**：
- 整体透明度：`0.5`
- 不可点击

**光标变化**：
- 光标变为 `not-allowed`

#### 4.2.4 聚焦状态（Focus）

**边框效果**：
- 显示聚焦边框
- 边框颜色：强调色（`#0078D4`）
- 边框宽度：`2px`
- 边框样式：`solid`

**发光效果**：
- 显示外发光效果
- 发光颜色：强调色（`#0078D4`）
- 发光半径：`4px`
- 发光透明度：`0.5`

## 五、材质效果

### 5.1 云母材质（Mica）

**特点**：
- 将背景与应用程序对齐创建颜色层次
- 区分应用程序的多个打开实例
- 性能优于亚克力材质

**应用场景**：
- 窗口标题栏
- 主窗口背景
- 设置页面背景

**实现方式**：
- 使用桌面壁纸的颜色
- 添加半透明模糊效果
- 透明度：`0.8-0.9`

### 5.2 亚克力材质（Acrylic）

**特点**：
- 半透明和模糊效果
- 显示窗口后面的内容
- 添加噪点纹理

**应用场景**：
- 弹出菜单
- 上下文菜单
- 临时窗口

**实现参数**：
- 模糊半径：`20-30px`
- 透明度：`0.6-0.8`
- 噪点纹理：`0.02`

### 5.3 标签材质（Tabbed）

**特点**：
- 对Acrylic和Mica的补充
- 截屏墙纸的严重模糊版本
- 没有固定的色调

**应用场景**：
- 标签页背景
- 分页容器

## 六、视觉效果

### 6.1 发光效果（Reveal Effect）

**特点**：
- 鼠标位置周围显示光晕
- 增强交互反馈
- 提升用户体验

**参数**：
- 光晕颜色：强调色（`#0078D4`）
- 光晕半径：`20-40px`
- 光晕透明度：`0.1-0.3`
- 动画时长：`150ms`

### 6.2 聚焦效果（Focus Effect）

**特点**：
- 键盘导航时显示
- 显示当前聚焦元素
- 提升可访问性

**参数**：
- 边框颜色：强调色（`#0078D4`）
- 边框宽度：`2px`
- 发光半径：`4px`
- 发光透明度：`0.5`

### 6.3 悬停效果（Hover Effect）

**特点**：
- 鼠标悬停时显示
- 背景色变化
- 轻微动画过渡

**参数**：
- 背景色变化：透明 → `#3D3D3D`（深色主题）
- 动画时长：`150ms`
- 动画曲线：`ease-out`

## 十四、设计令牌（Design Tokens）

### 14.1 令牌概念

设计令牌是一组命名的变量，用于定义主题的各种属性，如颜色、字体、间距等。

### 14.2 令牌优势

- **一致性**：确保所有组件使用相同的设计值
- **可维护性**：修改一处即可全局生效
- **可定制性**：方便主题切换和定制
