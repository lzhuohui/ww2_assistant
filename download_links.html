📱 Flet APK构建环境下载链接

1. Flutter SDK（必须）

国内镜像（推荐，速度快）：
Flutter中文网
直接下载地址
官方地址（可能需要VPN）：
Flutter官方下载
安装步骤：
  1. 下载 flutter_windows_3.16.0-stable.zip
  2. 解压到 C:\flutter
  3. 将 C:\flutter\bin 添加到系统Path环境变量
  4. 设置国内镜像环境变量（推荐）

2. Java JDK（必须）

Oracle JDK（官方）：
Oracle JDK下载
OpenJDK（免费开源）：
Eclipse Temurin OpenJDK
Zulu OpenJDK
安装步骤：
  1. 下载JDK 11或更高版本
  2. 运行安装程序
  3. 设置JAVA_HOME环境变量指向JDK安装目录
  4. 将 %JAVA_HOME%\bin 添加到系统Path环境变量

3. Android Studio（必须）

官方下载：
Android Studio下载
安装步骤：
  1. 下载Android Studio安装程序
  2. 运行安装程序，选择以下组件：
    • Android SDK
    • Android SDK Platform
    • Android Virtual Device（可选）
  3. 安装完成后，打开Android Studio
  4. 点击 Configure -> SDK Manager
  5. 安装 Android SDK Platform 33 或更高版本
  6. 安装 Android SDK Build-Tools 33.0.0 或更高版本
  7. 设置环境变量 ANDROID_HOME = C:\Users\[用户名]\AppData\Local\Android\Sdk

🔥 一键安装脚本

在项目目录中运行以下批处理文件：
  • 环境安装指南.bat - 完整的安装指南
  • setup_flutter_env.bat - 设置Flutter国内镜像
  • setup_android_env.bat - 设置Android SDK环境变量
  • verify_environment.bat - 验证环境安装
  • accept_licenses.bat - 接受Android许可

⚠️ 注意事项

  • 安装过程中可能需要重启计算机
  • 环境变量设置后需要重启命令行窗口才能生效
  • Android Studio第一次启动可能需要下载组件，请保持网络连接
  • Flutter SDK约1.5GB，Android Studio约1GB，请确保有足够磁盘空间
  • 建议使用国内镜像加速下载

🔧 验证安装

安装完成后，运行以下命令验证： 
java -version
flutter --version
adb --version
flutter doctor

🚀 开始构建

环境安装完成后，运行： 
python build_apk_simple.py
或 
python build_apk_full.py
