## 1. 折叠卡片组件 (CollapsibleCard)

### 1.1 文件位置
`前端/新界面_v2/表示层/组件/复合/折叠卡片.py`

### 1.2 核心状态变量

| 变量名 | 类型 | 初始值 | 用途 |
|--------|------|--------|------|
| `loaded` | List[bool] | [False] | 控件是否已加载 |
| `is_enabled` | List[bool] | [enabled] | 功能是否启用 |
| `switch_state` | List[bool] | [enabled] | 开关状态 |
| `control_dict` | Dict[str, ft.Control] | {} | 控件实例字典 |
| `current_values` | Dict[str, str] | {} | 当前值字典 |
| `initial_values` | Dict[str, str] | {} | 初始值字典 |

### 1.3 核心方法

| 方法名 | 参数 | 返回值 | 功能 |
|--------|------|--------|------|
| `handle_switch_toggle()` | 无 | 无 | 切换开关状态，调用on_save |
| `update_card_state()` | 无 | 无 | 更新卡片视觉状态（透明度、颜色） |
| `load_controls()` | 无 | 无 | 加载控件，调用on_expand回调 |
| `collapse()` | 无 | 无 | 折叠卡片，销毁控件，调用on_collapse回调 |
| `destroy_controls()` | 无 | 无 | 销毁控件，清空control_dict |
| `unload_options_only()` | 无 | 无 | 仅卸载下拉框选项，不销毁控件 |
| `get_values()` | 无 | Dict | 获取所有控件当前值 |
| `set_values(value_dict)` | Dict | 无 | 设置控件值 |
| `get_switch_state()` | 无 | bool | 获取开关状态 |
| `set_switch_state(state)` | bool | 无 | 设置开关状态 |
| `set_subtitle(new_subtitle, is_expanded)` | str, bool | 无 | 设置副标题 |

### 1.4 挂载到container的方法

```python
container.get_values = get_values
container.set_values = set_values
container.load_controls = load_controls
container.is_loaded = lambda: loaded[0]
container.get_switch_state = get_switch_state
container.set_switch_state = set_switch_state
container.destroy_controls = destroy_controls
container.unload_options_only = unload_options_only
container.collapse = collapse
container.set_subtitle = set_subtitle
```

### 1.5 关键行为

1. **点击左侧区域**：切换开关状态
2. **点击右侧区域（未加载时）**：加载控件
3. **开关关闭时**：卡片半透明，控件禁用
4. **开关打开时**：卡片正常，控件可用

### 2.3 卡片切换逻辑

```python
def handle_expand():
    if current_loaded_card[0] and current_loaded_card[0] != card:
        if current_loaded_card[0].is_loaded():
            current_loaded_card[0].unload_options_only()  # 只卸载选项
    current_loaded_card[0] = card
```

### 2.4 配置保存逻辑

```python
def handle_save(config_key: str, value: str):
    if card_id not in card_data:
        card_data[card_id] = {}
    card_data[card_id][config_key] = value
    if save_callback:
        save_callback(card_id, config_key, value)
```

### 2.5 初始化加载逻辑

1. 从ConfigService获取已保存的enabled状态
2. 如果没有保存，使用默认值并保存
3. 从ConfigService获取已保存的控件值
4. 如果没有保存，使用默认值并保存

## 4. 账号界面 (AccountPage)

### 4.1 文件位置
`前端/新界面_v2/表示层/界面/账号界面.py`

### 4.2 特殊状态管理

```python
class AccountPage:
    授权数量 = USER_AUTHORIZED_COUNT  # 15
    当前参与数量 = 0
    账号开关状态 = {}
```

### 4.3 卡片配置

- 15个账号卡片，card_id格式：`account_01` 到 `account_15`
- 第一个账号默认类型为"主帅"，其他为"副帅"

### 4.4 副标题逻辑（核心差异）

| 状态 | 折叠时副标题 | 展开时副标题 |
|------|-------------|-------------|
| 开关关闭 | "{name} (未参与), 设置请进入 >>" | "未参与挂机" |
| 开关打开但信息不完整 | "{name} (信息不完整), 设置请进入 >>" | "缺少: 名称/账号/密码" |
| 开关打开且信息完整 | "{name} (参与挂机), 设置请进入 >>" | "已配置: {name}" |

### 4.5 参与状态计算

```python
def can_participate(enabled: bool, name: str, account: str, password: str) -> bool:
    return enabled and bool(name) and bool(account) and bool(password)
```

### 4.6 授权限制逻辑

```python
def update_subtitle_and_count(...):
    if new_can and not old_can:
        if AccountPage.当前参与数量 >= AccountPage.授权数量:
            return False  # 拒绝开启
        AccountPage.当前参与数量 += 1
    elif not new_can and old_can:
        AccountPage.当前参与数量 -= 1
```

### 4.7 卡片切换逻辑

**账号界面没有卡片切换逻辑！**
- 不销毁控件
- 不卸载选项
- 每个卡片独立展开/折叠

### 4.8 展开时回调

```python
def handle_expand():
    # 更新副标题为展开状态格式
    update_subtitle_and_count(card_id, card, current_enabled, current_name, current_account, current_password, True)
```

### 4.9 折叠时回调

```python
def handle_collapse():
    # 更新副标题为折叠状态格式
    update_subtitle_and_count(card_id, card, current_enabled, current_name, current_account, current_password, False)
```

### 5.3 色块重新创建逻辑

```python
def update_theme_blocks():
    display_theme_name = "浅色" if current_theme[0] == "light" else "深色"
    selected_color = next((item["value"] for item in theme_color_list if item["name"] == display_theme_name), "#1A1A2E")
    theme_block_container.content = ThemeColorBlock.create_group(
        color_list=theme_color_list,
        selected_color=selected_color,
        on_select=handle_theme_switch,
        config=config,
    )
    if theme_block_container.page:
        theme_block_container.page.update()
```

### 5.4 主题切换逻辑

```python
def handle_theme_switch(color_value: str):
    for item in theme_color_list:
        if item["value"] == color_value:
            current_theme[0] = item["name"]
            theme_name = "light" if item["name"] == "浅色" else "dark"
            config.switch_theme(theme_name)
            if save_callback:
                save_callback("个性化", "theme", theme_name)
            update_theme_blocks()  # 重新创建色块
            break
```

### 5.5 卡片切换逻辑

**个性化界面没有卡片切换逻辑！**
- 不销毁控件
- 通过重新创建色块组件更新选中状态
