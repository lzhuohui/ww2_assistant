# 前端项目结构说明

## 目录结构

```
前端/
├── 入口/                  # 应用入口
│   └── main.py            # 主入口文件
├── 页面/                  # 页面模块
│   ├── 设置页面/          # 设置相关页面
│   ├── 监控页面/          # 监控相关页面
│   └── 关于页面/          # 关于页面
├── 组件/                  # 可复用组件
│   ├── 公共组件/          # 通用组件
│   ├── 业务组件/          # 业务特定组件
│   └── 浮窗组件/          # 浮窗相关组件
├── 服务/                  # 服务层
│   ├── 配置服务/          # 配置管理
│   ├── 监控服务/          # 监控服务
│   └── 数据服务/          # 数据处理
├── 工具/                  # 工具函数
│   ├── 通用工具/          # 通用工具函数
│   └── 业务工具/          # 业务特定工具
├── 资源/                  # 资源文件
│   ├── 图片/              # 图片资源
│   ├── 字体/              # 字体资源
│   └── 主题/              # 主题配置
└── 配置/                  # 配置文件
    ├── 默认配置/          # 默认配置
    └── 用户配置/          # 用户配置
```

## 运行方式

1. 安装依赖：
   ```
   pip install flet
   ```

2. 运行应用：
   ```
   python 入口/main.py
   ```

## 功能说明

1. **设置页面**：配置游戏路径、设备ID、脚本参数等
2. **监控页面**：显示运行状态、查看运行日志
3. **关于页面**：显示应用信息和使用说明

## 开发指南

1. **添加新页面**：在`页面`目录下创建新的页面目录和文件
2. **添加新组件**：在`组件`目录下创建可复用组件
3. **添加新服务**：在`服务`目录下创建服务模块
4. **添加新工具**：在`工具`目录下创建工具函数

## 技术栈

- **前端框架**：Flet (Python)
- **UI组件**：Flet内置组件
- **语言**：Python 3.10+

## 注意事项

1. 确保设备已连接并开启ADB调试
2. 确保游戏路径配置正确
3. 运行前请先配置必要的参数
