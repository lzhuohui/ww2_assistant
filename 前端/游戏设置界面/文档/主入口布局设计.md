## 七、交互行为

### 7.1 导航切换

| 操作 | 行为 |
|------|------|
| 点击导航项 | 切换右侧内容区 |
| 切换时 | 销毁当前内容区卡片（根据策略） |
| 动画 | 150ms ease-out |

### 7.2 卡片展开/折叠

| 操作 | 行为 |
|------|------|
| 点击卡片左侧 | 切换开关状态 |
| 点击卡片右侧 | 展开/折叠控件 |
| 展开时 | 折叠同组其他卡片（根据策略） |

## 八、布局代码结构

```python
# 主布局结构（简化版 - 两大区域）
main_layout = ft.Row([
    # 左侧导航栏（区域一 + 区域二）
    ft.Container(
        width=220,
        content=ft.Column([
            # 区域一：用户信息区 (100px)
            UserInfoCard.create(config=config),
            ft.Container(height=10),  # 间距
            # 区域二：导航菜单区
            ft.Column([
                *nav_buttons,  # 导航按钮列表
            ], expand=True),
        ]),
        bgcolor=theme_colors.get("bg_primary"),
    ),
    # 分隔线
    ft.VerticalDivider(width=1, color=theme_colors.get("border")),
    # 右侧内容区
    ft.Container(
        expand=True,
        content=ft.Column([
            # 标题栏 (40px)
            title_bar,
            # 内容区 (自适应)
            content_section,
        ]),
    ),
], expand=True, spacing=0)
```
