### 共识 #004：三种销毁策略

| 项目 | 内容 |
|------|------|
| **达成时间** | 2026-03-25 |
| **问题描述** | 不同配置区对卡片切换时的处理需求不同 |
| **讨论方案** | A：统一销毁策略<br>B：支持多种策略 |
| **最终决策** | 支持三种销毁策略：none/unload_options/destroy_controls |
| **决策原因** | 满足不同配置区的特殊需求，保持灵活性 |

| 策略 | 适用场景 | 行为 |
|------|----------|------|
| `none` | 账号配置区、个性化配置区 | 不销毁控件 |
| `unload_options` | 系统配置区、策略配置区 | 只卸载下拉框选项 |
| `destroy_controls` | 预留 | 销毁所有控件 |
