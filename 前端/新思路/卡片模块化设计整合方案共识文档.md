## 一、背景与目标

### 1.1 背景
系统设置页面包含三个卡片：
- 基础设置卡片（使用标签下拉框）
- 主题设置卡片（使用主题色块）
- 调色板设置卡片（使用主题色块）

### 1.2 目标
将三个卡片的通用逻辑提取并集成到通用卡片组件中，实现"调用通用卡片后通过简单路径处理即可实现各种卡片效果"的目标。

---

## 二、核心共识

### 2.1 技术方案共识

#### 方案选择
采用**配置驱动 + 工厂模式**的方案。

#### 核心优势
1. **代码复用率提升**：从当前的60%提升到90%以上
2. **配置驱动**：所有卡片通过配置文件定义，易于维护
3. **统一接口**：开发者只需学习一套API
4. **易于扩展**：新增卡片类型只需扩展配置和工厂方法
5. **降低耦合**：卡片逻辑与界面分离

## 四、控件工厂设计

### 4.1 工厂类结构

```python
class ControlFactory:
    """控件工厂 - 根据配置创建控件"""
    
    @staticmethod
    def create_controls(config, card_config, config_manager):
        """根据配置创建控件列表"""
        card_type = card_config.get("card_type", "standard")
        
        if card_type == "standard":
            return ControlFactory._create_standard_controls(config, card_config, config_manager)
        elif card_type == "color_blocks":
            return ControlFactory._create_color_blocks(config, card_config, config_manager)
        else:
            return []
```

### 4.2 控件创建逻辑

#### 标准控件创建
- 支持类型：dropdown（下拉框）、input（输入框）、switch（开关）
- 根据配置动态创建控件
- 从ConfigManager获取配置值

#### 色块控件创建
- 支持类型：theme（主题色块）、palette（调色板色块）
- 支持取消选择功能
- 自动管理选中状态

## 五、通用卡片扩展

### 5.1 新增方法

```python
@staticmethod
def create_from_config(
    config: 界面配置,
    card_name: str,
    config_manager: ConfigManager,
    on_value_change: Callable = None,
    **kwargs
):
    """根据配置创建卡片"""
    card_config = config_manager.get_card_config(card_name)
    controls = ControlFactory.create_controls(config, card_config, config_manager)
    
    return UniversalCard.create(
        config=config,
        title=card_config.get("title"),
        icon=card_config.get("icon"),
        subtitle=card_config.get("subtitle"),
        controls=controls,
        controls_per_row=card_config.get("controls_per_row", 1),
        **kwargs
    )
```

## 十、错误处理机制

### 10.1 错误处理策略

1. **配置验证**：验证配置完整性和正确性
2. **容错处理**：使用默认值替代错误配置
3. **错误隔离**：单个卡片错误不影响其他卡片
4. **友好提示**：向用户显示友好的错误信息

### 10.2 错误处理实现

```python
@staticmethod
def create_from_config(config, card_name, config_manager, **kwargs):
    """带错误处理的创建方法"""
    try:
        # 配置验证
        card_config = UniversalCard._validate_config(
            config_manager.get_card_config(card_name)
        )
        
        # 控件创建（带错误处理）
        controls = UniversalCard._create_controls_safe(
            config, card_config, config_manager
        )
        
        # 卡片创建
        return UniversalCard.create(...)
        
    except ConfigNotFoundError as e:
        return UniversalCard._create_error_card(config, f"配置错误: {card_name}")
    except Exception as e:
        return UniversalCard._create_error_card(config, "系统错误")
```
