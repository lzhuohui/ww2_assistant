# 主界面

## 职责

- 主入口：应用启动入口
- 用户信息卡片：显示用户状态（试用/授权）、剩余天数
- 导航栏：提供各功能模块的导航入口
- 主框架：左右分栏布局

## 依赖规则

- ✅ 可依赖：设置页面、组件、零件、工具、配置
- ❌ 禁止：被其他层依赖

## 模块示例

- 主界面.py：主入口，组装用户信息卡片和导航栏
- 用户信息卡片.py：显示用户状态
- 导航栏.py：导航按钮列表
