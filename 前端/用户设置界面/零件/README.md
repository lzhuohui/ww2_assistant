# 零件

## 职责

- 提供基础UI元素
- 实现单一功能
- 最小可复用单元

## 依赖规则

- ✅ 可依赖：工具、配置
- ❌ 禁止：依赖组件、设置页面、主界面

## 模块示例

| 模块 | 功能描述 |
|------|---------|
| 头像.py | 用户头像显示 |
| 导航按钮.py | 导航按钮功能 |
| 标签输入框.py | 标签+输入框组合 |
| 标签下拉框.py | 标签+下拉框组合 |
| 卡片容器.py | 卡片外轮廓、阴影 |
| 分割线.py | 垂直分割线 |
| 主题色块.py | 主题预览色块 |

## 设计原则

1. 零件应该是最小可复用单元
2. 零件应该职责单一
3. 零件应该可独立使用
