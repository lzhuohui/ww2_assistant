# 配置管理逻辑设计文档

## 概述

本文档描述Python应用程序的配置管理最佳实践，实现自动检测和初始化配置文件，支持多套配置方案管理。

## 文件结构

```
前端/配置/
├── 默认配置.json           # 系统默认配置
├── 用户配置.json           # 当前使用的配置（游戏控制读取）
├── 配置管理器.py           # 配置管理逻辑
└── 配置方案/               # 用户保存的多套配置
    ├── 快速挂机.json
    ├── 稳定挂机.json
    └── 自定义方案.json
```

## 配置优先级

```
用户配置 > 卡片配置 > 代码默认值
```

## 自动配置管理流程

```
程序启动
    ↓
检查用户配置文件
    ↓
├─ 不存在 → 自动创建默认配置
├─ 存在但为空 → 自动创建默认配置
├─ 存在但损坏 → 自动创建默认配置（备份旧文件）
└─ 存在且有效 → 读取用户配置
    ↓
游戏控制核心读取配置
```

## 多套配置方案管理

```

用户检查/修改配置
    ↓
```

## 配置管理器实现要点

### 1. 加载用户配置

```python
def _load_user_config(self) -> Dict[str, Any]:
    """加载用户配置（自动初始化）"""
    if not self.user_config_file.exists():
        return self._create_default_config()
    
    try:
        with open(self.user_config_file, 'r', encoding='utf-8') as f:
            content = f.read().strip()
            if not content:
                return self._create_default_config()
            return json.loads(content)
    except Exception as e:
        print(f"加载用户配置失败: {e}")
        return self._create_default_config()
```

### 2. 创建默认配置

```python
def _create_default_config(self) -> Dict[str, Any]:
    """创建默认配置"""
    if self.default_config_file.exists():
        try:
            with open(self.default_config_file, 'r', encoding='utf-8') as f:
                default_config = json.load(f)
            self.user_config = default_config
            self._save_user_config()
            print("已自动创建默认配置文件")
            return default_config
        except Exception as e:
            print(f"加载默认配置失败: {e}")
    return {}
```

### 5. 配置方案管理

```python
def get_scheme_list(self) -> List[str]:
    """获取所有配置方案列表"""
    scheme_dir = self.config_dir / "配置方案"
    if not scheme_dir.exists():
        scheme_dir.mkdir()
    return [f.stem for f in scheme_dir.glob("*.json")]

def load_scheme(self, scheme_name: str) -> Dict[str, Any]:
    """加载指定配置方案"""
    scheme_file = self.config_dir / "配置方案" / f"{scheme_name}.json"
    if scheme_file.exists():
        with open(scheme_file, 'r', encoding='utf-8') as f:
            return json.load(f)
    return {}

def save_scheme(self, scheme_name: str, config: Dict[str, Any]):
    """保存配置方案"""
    scheme_dir = self.config_dir / "配置方案"
    if not scheme_dir.exists():
        scheme_dir.mkdir()
    scheme_file = scheme_dir / f"{scheme_name}.json"
    with open(scheme_file, 'w', encoding='utf-8') as f:
        json.dump(config, f, ensure_ascii=False, indent=2)

def apply_scheme(self, scheme_name: str):
    """应用配置方案到用户配置"""
    scheme_config = self.load_scheme(scheme_name)
    if scheme_config:
        self.user_config = scheme_config
        self._save_user_config()
```
