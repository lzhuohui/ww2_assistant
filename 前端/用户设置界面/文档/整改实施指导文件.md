## 一、整改目标与范围

### 1.1 整改目标

| 目标层级 | 目标描述 | 可量化指标 |
|---------|---------|-----------|
| **核心目标** | 实现自底向上的模块化组合架构 | 模块层级清晰，职责单一 |
| **质量目标** | 统一代码规范，提高可维护性 | 代码规范符合率100% |
| **效率目标** | 简化界面模块代码，减少重复 | 界面代码减少70% |
| **扩展目标** | 支持配置驱动，便于功能扩展 | 新增卡片只需配置 |

### 1.2 整改范围

```
前端/用户设置界面/
├── 核心接口/          ← 阶段1：完善Provider
```

```python
# 规范化模板（以输入框为例）
# -*- coding: utf-8 -*-
"""模块名称：输入框 | 设计思路 | 模块隔离原则"""
import flet as ft

# *** 用户指定变量 - AI不得修改, 变量值必须生效 ***
USER_WIDTH = 200  # 默认输入框宽度
USER_HEIGHT = 32  # 默认输入框高度
# *********************************

# 默认值常量 - 供调用者获取
DEFAULT_WIDTH = USER_WIDTH
DEFAULT_HEIGHT = USER_HEIGHT
DEFAULT_PLACEHOLDER = "请输入"

class Input:
    @staticmethod
    def create(
        value: str="",
        placeholder: str=DEFAULT_PLACEHOLDER,
        width: int=DEFAULT_WIDTH,
        height: int=DEFAULT_HEIGHT,
        on_change: Callable=None,
    ) -> ft.TextField:
        return ft.TextField(
            value=value,
            hint_text=placeholder,
            width=width,
            height=height,
            on_change=on_change,
        )

if __name__ == "__main__":
    ft.run(lambda page: page.add(Input.create()))
```

```python
# 任务3.1：创建SettingsPanel
# 文件：组件模块/设置面板.py
# 功能：合并SettingsContainer和GenericFunctionContainer

class SettingsPanel:
    """设置面板 - 统一入口"""
    
    @staticmethod
    def create(
        config: 界面配置,
        title: str="设置",
        icon: str="SETTINGS",
        card_names: List[str]=None,  # 配置驱动
        cards: List[ft.Control]=None,  # 自定义卡片
        expand: bool=False,
        **kwargs
    ) -> ft.Container:
        final_cards = []
        
        # 1. 从配置创建标准卡片
        if card_names:
            for card_name in card_names:
                card = SettingsPanel._create_card_from_config(card_name)
                if card:
                    final_cards.append(card)
        
        # 2. 添加自定义卡片
        if cards:
            final_cards.extend(cards)
        
        # 3. 使用通用布局
        return GenericFunctionContainer.create(
            config=config,
            title=title,
            icon=icon,
            cards=final_cards,
            expand=expand,
            **kwargs
        )
```
