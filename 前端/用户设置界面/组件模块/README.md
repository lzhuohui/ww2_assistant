# 设置页面

## 职责

- 各功能设置页面的实现
- 配置驱动渲染
- 实时保存配置

## 依赖规则

- ✅ 可依赖：组件、零件、工具、配置
- ❌ 禁止：依赖主界面

## 模块列表

| 模块 | 功能描述 |
|------|---------|
| 系统设置.py | 挂机模式、指令速度、尝试次数、清缓限量 |
| 策略设置.py | 建筑速建、资源速产、策点保留 |
| 任务设置.py | 任务相关配置 |
| 建筑设置.py | 建筑相关配置 |
| 集资设置.py | 集资相关配置 |
| 账号设置.py | 账号管理配置 |
| 打扫设置.py | 打扫功能配置 |
| 打野设置.py | 打野功能配置 |
| 个性化设置.py | 主题、调色板等 |
| 关于设置.py | 版本信息、联系方式 |

## 设计原则

1. 配置驱动：通过配置文件定义卡片结构
2. 实时保存：配置变化立即保存
3. 独立调试：每个页面可独立运行
