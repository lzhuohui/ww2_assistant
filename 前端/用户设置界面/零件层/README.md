# 零件层

## 职责

- 提供基础UI元素
- 实现单一功能
- 最小可复用单元

## 依赖规则

- ✅ 可依赖：工具层、配置层
- ❌ 禁止：依赖组件层、页面层

## 模块示例

- 卡片容器.py：卡片外轮廓、阴影、边框
- 分割线.py：垂直分割线
- 标签文本.py：文本显示
- 自定义下拉框.py：下拉选择功能
- 标签下拉框.py：标签+下拉框组合
- 标签输入框.py：标签+输入框组合
- 图标标题.py：图标+标题+分割线组合
- 导航按钮.py：导航按钮功能
- 主题色块.py：主题预览色块

## 设计原则

1. 零件应该是最小可复用单元
2. 零件应该职责单一
3. 零件应该可独立使用
4. 零件不应该依赖其他零件（组合除外）
