# Flet界面实例测试

## 说明

本目录包含Flet界面的实例测试文件，用于展示实际的Flet界面效果。

## 文件列表

| 文件 | 说明 |
|------|------|
| `test_flet界面实例.py` | 基础界面测试（按钮、输入框、进度条、卡片） |
| `test_配置界面实例.py` | 配置界面测试（游戏辅助工具配置界面） |
| `test_监控界面实例.py` | 监控界面测试（实时监控面板） |
| `运行界面测试.bat` | 批处理启动脚本 |

## 运行方式

### 方式1：直接运行Python文件

```bash
# 运行基础界面测试
python test_flet界面实例.py

# 运行配置界面测试
python test_配置界面实例.py

# 运行监控界面测试
python test_监控界面实例.py
```

### 方式2：使用批处理脚本

双击运行 `运行界面测试.bat`，然后选择要运行的测试：
- 选项1：基础界面测试
- 选项2：配置界面测试
- 选项3：监控界面测试
- 选项4：依次运行全部测试

### 方式3：在VS Code中运行

1. 打开要运行的Python文件
2. 按 `F5` 或点击运行按钮
3. 选择 "Python: 当前文件"

## 界面说明

### 基础界面测试

包含以下功能：
- **按钮交互测试**：点击计数、重置功能
- **输入框测试**：输入名字并显示问候语
- **进度条测试**：模拟进度加载过程
- **卡片组件展示**：显示信息卡片

### 配置界面测试

模拟游戏辅助工具的配置界面：
- 游戏路径设置
- 设备ID设置
- 连接超时设置
- 脚本延迟设置
- 运行模式选择
- 主题切换

### 监控界面测试

模拟实时监控面板：
- 运行状态显示
- 运行时间统计
- 点击次数统计
- 实时数据图表
- 运行日志显示
- 开始/停止控制

## 注意事项

1. **界面位置**：Flet界面可能会出现在屏幕中央或任务栏，请注意查看
2. **关闭方式**：点击窗口右上角的X按钮关闭界面
3. **终端保持**：运行界面时终端窗口需要保持打开状态
4. **多实例**：可以同时运行多个界面实例

## 常见问题

### Q: 界面没有显示？

A: 请检查：
1. 终端是否有错误信息
2. 任务栏是否有新窗口
3. 是否被其他窗口遮挡

### Q: 如何关闭界面？

A: 点击界面窗口右上角的X按钮即可关闭。

### Q: 可以同时运行多个界面吗？

A: 可以，每个界面都是独立的进程。
