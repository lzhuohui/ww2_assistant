## 迁移指南

### 1. 导入语句变更
```python
# 旧代码（使用原始版本）
from 前端.用户设置界面.界面模块.用户界面 import UserInfoCard

# 新代码（使用最终版本）
from 前端.用户设置界面.界面模块.可编辑统一风格用户界面 import UserInfoCard
```

### 2. 使用方式
```python
# 创建用户卡片（接口完全兼容）
card = UserInfoCard.create(
    username="用户",
    expire_days=30,
    is_registered=True
)

# 可选：添加头像文字变化回调
def on_avatar_text_change(new_text):
    print(f"头像文字修改为: {new_text}")

card = UserInfoCard.create(
    username="用户",
    expire_days=30,
    on_avatar_text_change=on_avatar_text_change
)

# 编程控制头像
card.set_avatar_text("新文字")  # 设置头像文字
card.get_avatar_text()         # 获取头像文字
card.set_avatar_image("url")   # 设置头像图片
```

### 3. 交互功能
- **单击/长按**: 显示"双击可编辑"提示
- **双击**: 进入编辑状态
- **编辑状态**: 自动选中文字，可输入
- **保存方式**: 按Enter或点击其他地方
- **文字限制**: 只允许单个汉字
