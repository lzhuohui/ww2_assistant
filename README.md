# 二战风云辅助工具

## 项目简介

基于Python和Flet框架开发的手游辅助工具，用于自动化游戏操作。

## 项目结构

```
二战风云/
├── .trae/              # Trae IDE配置
│   ├── rules/          # 规则文件
│   └── skills/         # 技能文件
├── AI产物/             # AI生成的文件和工具
│   ├── 文档/           # 文档类文件
│   ├── 工具/           # 工具脚本
│   └── ...
├── 前端/               # Flet前端代码
│   ├── 入口/           # 应用入口
│   ├── 页面/           # 页面模块
│   ├── 配置/           # 配置文件
│   │   ├── requirements.txt    # Python依赖列表
│   │   └── activate_env.bat    # 虚拟环境激活脚本
│   └── ...
├── venv/               # Python虚拟环境
├── .gitignore          # Git忽略配置
├── project_rules.md    # 项目规则索引
└── README.md           # 本文件
```

## 快速开始

### 1. 激活虚拟环境

```bash
前端/配置/activate_env.bat
```

### 2. 运行前端应用

```bash
python 前端/入口/main.py
```

### 3. 运行测试

```bash
python AI产物/工具/脚本/脚本_20260308_Flet环境测试_v1.0.0.py
```
