## 使用方法

### 1. 导入最终版本
```python
# 导入用户界面
from 前端.用户设置界面.界面模块.可编辑统一风格用户界面 import UserInfoCard

# 如果需要直接使用头像
from 前端.用户设置界面.单元模块.可编辑可靠头像 import EditableReliableAvatar
```

### 2. 创建用户卡片（接口完全兼容）
```python
# 基本使用（与原始界面完全一致）
card = UserInfoCard.create(
    username="帅气的用户",
    is_registered=True,
    expire_days=30
)

# 添加头像文字变化回调
def on_avatar_text_change(new_text):
    print(f"头像文字修改为: {new_text}")

card = UserInfoCard.create(
    username="用户",
    expire_days=30,
    on_avatar_text_change=on_avatar_text_change
)
```

### 3. 编程控制头像
```python
# 设置头像文字
card.set_avatar_text("新文字")

# 获取头像文字
current_text = card.get_avatar_text()
print(f"当前头像文字: {current_text}")

# 设置头像图片
card.set_avatar_image("https://example.com/avatar.jpg")
```

### 4. 直接使用头像组件
```python
# 创建独立的可编辑头像
avatar = EditableReliableAvatar.create(
    size=70,        # 头像直径
    text="帅",      # 显示文字
    show_glow=True, # 显示光晕
    on_text_change=lambda new_text: print(f"文字修改: {new_text}")
)

# 控制方法
avatar.set_text("爱")      # 设置文字
avatar.get_text()         # 获取文字
avatar.set_image("url")   # 设置图片
avatar.get_editing()      # 获取编辑状态
```
